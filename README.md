# Portfólio — Nicholas Rieger

Meu portfólio pessoal como desenvolvedor front-end, criado para apresentar meu trabalho de forma clara, visualmente cuidadosa e com foco em experiência no celular.

O objetivo deste site é reunir meus principais projetos, mostrar um pouco da minha proposta como desenvolvedor e facilitar o contato para oportunidades profissionais e projetos freelance.

## Preview

O site apresenta:

- Hero de apresentação
- Seção de projetos em destaque
- Seção sobre mim
- Seção de contato
- Links diretos para projetos publicados e código no GitHub

## Projetos destacados

### Studio JX
Landing page desenvolvida para um estúdio de estética, com foco em apresentação da marca, organização dos serviços e contato mais direto pelo WhatsApp.

### Cardápio Digital
Projeto de cardápio online pensado para tornar o pedido mais simples, visualmente organizado e confortável de usar no celular.

## Tecnologias utilizadas

- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome

## Estrutura do projeto

```bash
assets/
styles/
  input.css
  output.css
index.html
package.json
package-lock.json
Como executar localmente

Se quiser abrir o projeto localmente:

Clone este repositório
Instale as dependências
Rode o Tailwind em modo watch
Abra o index.html no navegador
Instalação
npm install
Compilar o CSS com Tailwind
npx @tailwindcss/cli -i ./styles/input.css -o ./styles/output.css --watch
```
